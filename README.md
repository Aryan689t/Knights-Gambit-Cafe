# Knights Gambit Cafe

A beginner web development project built using HTML and CSS.
This project is a themed café landing page created while learning basic web development concepts such as:
- HTML page structure
- CSS styling
- Flexbox layout
- Navigation bar creation

## Features

- Navigation bar
- Flexbox layout
- Themed café landing page
- Simple call-to-action button

## Technologies Used

- HTML
- CSS
